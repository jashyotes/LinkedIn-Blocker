# LinkedIn Blocker - Version Notes & Reviewer Information

## Version 1.0.0

Initial release. Blocks all linkedin.com requests using declarativeNetRequest. Includes a typing challenge to prevent impulsive unblocking.

## Testing

No account or login needed. The extension works immediately on install:

1. Install the extension — linkedin.com is now blocked.
2. Click the toolbar icon to open the popup.
3. "Disable for 5 min" — presents a 10-20 character typing challenge. Completing it disables blocking for 5 minutes.
4. "Customized Disable" — presents a 35 character typing challenge and lets you specify a duration (1-60 minutes).
5. During the countdown, clicking the timer button re-enables blocking early.

Copy/paste is disabled in the challenge input to require manual typing.
